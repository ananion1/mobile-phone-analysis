# Mobile Phone Price Analysis

## Overview

This project explores a mobile phone dataset to understand how different smartphone features influence market prices. Using Python and data analysis libraries, the dataset was cleaned, analyzed, and visualized to uncover patterns and relationships between various specifications and pricing.

## Objectives

* Analyze smartphone pricing trends.
* Identify the features that have the greatest impact on price.
* Visualize relationships between specifications and market value.
* Extract meaningful insights through exploratory data analysis (EDA).

## Features Analyzed

* RAM
* Internal Storage
* Battery Capacity
* Camera Specifications
* Processor Details
* Display Features
* Brand Information
* Mobile Phone Price

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Examined correlations between smartphone specifications and price.
* Compared how different features contribute to overall device cost.
* Generated visualizations to identify pricing trends and feature importance.
* Derived data-driven conclusions from the analyzed dataset.
